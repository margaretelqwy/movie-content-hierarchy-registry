# movie-content-hierarchy-registry
该项目围绕在线电影与电视剧网站的内容层级结构展开整理，对首页模块分布逻辑、导航层级关系以及播放页信息区块安排进行系统记录。通过结构化文字归档建立资料库，用于分析不同影视平台在页面规划与内容组织方面的变化趋势。仓库仅保存公开页面资料，仅供研究与信息整理参考使用，不涉及影视资源文件或在线播放功能。
